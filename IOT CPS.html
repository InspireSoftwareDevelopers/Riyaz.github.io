Introduction:


"Things" in IOT, are not "physical" components in cyber physical systems. Things are those embedded systems that can compute and communicate (Cyber) Physical components are those which to be monitored (Human Body), which need to be tracked (Logistics), which need to be guided (Vehicles), which need to be controlled (Industrial Drivers).
